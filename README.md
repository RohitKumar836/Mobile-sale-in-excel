# Mobile Sales Data Analysis Dashboard

## Project Overview

This project focuses on analyzing mobile sales data and building an interactive dashboard using Excel to extract meaningful business insights.

---

## Key Metrics

* **Total Revenue:** $4,639,641
* **Total Units Sold:** 5,229
* **Total Customers:** 500
* **Average Price:** $889.9
* **Average Rating:** 4.0

---

## Dashboard Features

* 🔹 Revenue by Brand
* 🔹 Revenue by Model
* 🔹 Units Sold by Color
* 🔹 Revenue by Country
* 🔹 Units Sold by Storage Variant
* 🔹 Revenue by Payment Method

---

## 🛠 Tools & Technologies

* Microsoft Excel

* Data Cleaning
* Data Visualization

## Insights Generated

* Identified top-performing brands and models
* Analyzed customer buying patterns
* Found popular storage variants and colors

* Evaluated preferred payment methods
